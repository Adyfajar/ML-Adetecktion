# AnemiaScan (Deteksi Dini Anemia Non-Invasif Berbasis Web)

## Deskripsi

**AnemiaScan** adalah aplikasi berbasis web yang memudahkan deteksi dini anemia tanpa perlu pemeriksaan darah invasif. Proyek ini dibuat untuk membantu masyarakat mendeteksi anemia secara cepat, praktis, dan mandiri, sehingga pencegahan dan penanganan bisa dilakukan lebih awal.

Aplikasi ini menggabungkan:
- Machine Learning untuk menganalisis citra atau data pengguna dan mendeteksi risiko anemia
- Antarmuka web interaktif yang memudahkan pengguna melakukan skrining mandiri

Dengan AnemiaScan, deteksi anemia menjadi lebih mudah, cepat, dan dapat diakses oleh siapa saja.
